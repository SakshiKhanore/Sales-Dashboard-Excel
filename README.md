# Retail Sales Analysis Dashboard

**Tools:** Python (Pandas) · Microsoft Excel · Jupyter Notebook

## Project Overview

An end-to-end retail sales analysis project combining Python-based data processing 
with an interactive Excel dashboard to uncover revenue trends, customer behaviour 
patterns, and product performance insights across 1,000+ transactions.

The goal was to validate findings across two workflows — programmatic analysis in 
Python and spreadsheet-based dashboarding in Excel — ensuring consistency and 
accuracy of insights before presenting to stakeholders.

## Business Objective

- Identify monthly revenue trends and seasonal patterns
- Analyse sales performance by product category and gender
- Surface top-performing customers and high-value segments
- Validate insights across both Python and Excel workflows

## Dataset

- 1,000+ retail transactions
- Features: Transaction ID, Date, Customer ID, Gender, Age, 
  Product Category, Quantity, Price per Unit, Total Amount

## Tools & Technologies

- **Python (Pandas)** — data cleaning, filtering, aggregation, 
  revenue metrics computation
- **Microsoft Excel** — Pivot Tables, KPI cards, interactive 
  charts (Line, Bar, Donut), dashboard design
- **Jupyter Notebook** — exploratory data analysis and 
  programmatic validation

## Key Insights

- **Top category by revenue:** Beauty products contributed the 
  highest share of total revenue (~40%)
- **Peak sales month:** Revenue spiked consistently in Q1 and Q3, 
  suggesting seasonal buying patterns
- **Gender split:** Female customers accounted for a higher 
  transaction volume across most product categories
- **High-value customers:** Top 5 customers by spend identified 
  and visualised for targeted retention focus

## Dashboard Features (Excel)

- KPI cards: Total Revenue, Total Quantity, Average Order Value, 
  Total Transactions
- Monthly revenue trend (Line chart)
- Sales by product category (Bar chart)
- Gender distribution (Donut chart)
- Top 5 customers by revenue

## What I Learned

- Cross-validating insights across two different tools 
  (Python vs Excel) builds confidence in analytical outputs
- Pivot tables and programmatic aggregation produce consistent 
  results when data cleaning is done correctly
- Dashboard design choices (chart type, KPI placement) 
  significantly impact how insights are communicated to 
  non-technical stakeholders
